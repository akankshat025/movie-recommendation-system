
#  Movie Recommendation System

A content-based movie recommendation system built with Python and scikit-learn.

## Features
- Content-based recommendations using TF-IDF and cosine similarity
- Genre-based filtering
- Weighted recommendations combining similarity, rating, and popularity
- KMeans clustering of movies

## Dataset
Uses the TMDB Movies dataset — download it from
[Kaggle](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata)
and place it as `movies.csv` in the notebook directory.

## How to Run
1. Clone this repo
2. Install dependencies: `pip install -r requirements.txt`
3. Open `MovieRecSys.ipynb` in Jupyter or Google Colab
4. Update the dataset path in Cell 2 and run all cells

## Tech Stack
Python, NumPy, Pandas, scikit-learn, Matplotlib, Seaborn
